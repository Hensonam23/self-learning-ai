# Self-Learning AI

A from-scratch evolving AI that learns to [e.g., approximate numbers or play games] using evolutionary algorithms in Python.

## Setup
- Clone: `git clone https://github.com/Hensonam23/self-learning-ai.git`
- Install dependencies: `pip install numpy matplotlib`
- Run: `python evolve_ai.py`

## Features
- Evolutionary loop with mutation and crossover.
- Visualizes progress with plots.
